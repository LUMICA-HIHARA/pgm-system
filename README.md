# PGM.sys — 製品開発管理システム

商品開発・調達フローをゲート審査方式で管理するブラウザアプリです。

---

## アクセス

**URL：** `https://[your-username].github.io/[repo-name]/`

---

## ログイン情報

| アカウント | パスワード | 権限 |
|-----------|-----------|------|
| 開発進行管理（管理者） | `pgm-admin` | 全案件・全チーム閲覧、ユーザー管理 |
| 各チームユーザー | 登録時に設定 | 自チームの案件のみ |

> ⚠️ 本番運用前に管理者パスワードを変更してください（コード内 `hashSync('pgm-admin')` の部分）

## データ共有の運用方法

このシステムはブラウザのlocalStorageにデータを保存します。
**端末・ブラウザが異なるとデータは共有されません。**

### 推奨運用フロー

```
【作業開始時】
OneDriveの共有フォルダから最新の PGM_data_*.json を取得
→ アプリ上部「JSONインポート」で読み込む

【作業終了時】
アプリ上部「JSONエクスポート」でファイルを保存
→ OneDriveの共有フォルダに上書き保存
→ チームに「更新しました」と一言連絡
```

### 共有フォルダの推奨構成

```
OneDrive/
└── PGM_共有/
    ├── PGM_data_最新.json      ← 常に最新版
    ├── バックアップ/
    │   ├── PGM_data_20250115.json
    │   └── PGM_data_20250114.json
    └── デザイン素材/
        └── [案件別フォルダ]
```

## 技術仕様

- **フロントエンド：** HTML / CSS / Vanilla JavaScript（フレームワーク不使用）
- **データ保存：** ブラウザ localStorage
- **外部依存：** Google Fonts（Noto Sans JP）のみ
- **ファイル構成：** index.html 1ファイルのみ
